## üìò Course Taught

I taught **EGR101 ‚Äî Introduction to Programming for Engineers**.

The course focused on foundational computational and analytical skills using:

- MATLAB  
- Python  
- Microsoft Excel  

EGR101 serves as an early technical filter course within the engineering curriculum, establishing programming fundamentals, structured problem-solving habits, and analytical thinking skills that students carry forward into upper-level coursework.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## üß† Teaching Philosophy

My course was intentionally rigorous.

Students were expected to:

- Demonstrate conceptual understanding
- Apply programming to solve structured engineering problems
- Communicate technical reasoning clearly
- Develop discipline in analytical thinking
- Debug and validate their own work methodically

I was not an ‚Äúeasy A‚Äù instructor.

However, students who completed the course successfully demonstrated competency across multiple levels ‚Äî conceptual, computational, and applied.

The goal was never to filter students out.

The goal was to prepare them properly.
